# incident-response-lab

## Описание инцидента
Учебное расследование симулированной SSH brute-force атаки с последующим созданием подозрительного файла в системе Simply Linux (ALT-based).

## Цели проекта
- Практика расследования инцидентов в Linux-среде
- Документирование полного цикла: от обнаружения до отчёта
- Сбор портфолио для позиций SOC Analyst / Junior Security Specialist


## Технологии и навыки
- ОС: Simply Linux (ALT Linux)
- Инструменты: grep, SSH, ls, chmod, systemctl
- Навыки: Анализ логов (/var/log/secure), расследование инцидентов, документирование

## Для работодателя
Этот проект демонстрирует практические навыки, необходимые для начальной позиции в SOC:
- Умение работать с логами Linux
- Понимание цепочки атаки (MITRE ATT&CK: T1110 → T1078 → T1222)
- Навык структурированного документирования инцидентов
